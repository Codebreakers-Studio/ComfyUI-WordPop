============================================
  WORD POP - Word-by-Word Subtitle Generator
============================================

  QUICK START:

  1. Double-click INSTALL.bat  (one time setup)
     - Checks for Python (guides you if missing)
     - Downloads FFmpeg automatically
     - Installs AI packages

  2. Double-click Word-Pop.bat
     - Drop your audio/video file
     - Pick a style, hit enter
     - Done!

  OUTPUT FILES (in /output folder):

    _wordpop.mp4            Playable preview with subs
    _wordpop_overlay.webm   Transparent overlay for editors
    _wordpop.ass            Subtitle file for any editor

  The .webm overlay has transparency - drag it into
  Premiere / DaVinci / CapCut over your video and
  resize freely.

  REQUIREMENTS:
    - Windows 10/11
    - Python 3.10+  (INSTALL.bat will guide you)
    - Internet       (first run downloads AI model)

  STYLES: neon, clean, bold, minimal, fire, boxed
  Type 'styles' in Word-Pop to preview them.

  EXAMPLE:
    File: C:\Users\You\video.mp4
    Options: fire +large +karaoke

    That gives you: fire style, best accuracy,
    karaoke mode. Or just press Enter for defaults.

  TIPS:
    - First run downloads the AI model (~150MB)
    - Use +large for best accuracy (~3GB download)
    - +karaoke shows full lines with word highlight
    - Default pop mode shows one word at a time
